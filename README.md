## This project is a self-study school project. The project is designed upon Telco sensors use case description.

## Getting started
1. Open terminal
    1.1 - cd "project folder"
    1.2 - cd backend
    1.3 - nodemon server (The server will start automatically)
    1.4 - Open new terminal window
    1.5 - cd frontend
    1.6 - expo start (Starts the expo server)
    1.7 - change frontend api endpoint to your local ip address
    1.8 --> run on your mobile(Expo app /QR code) or emulator
    (have to install node modules inside frontend and backend folder as well)

### frontend- React Native(Expo framework)
### backend - Node.js/express.js/MongoDB
